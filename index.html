Team 36 AR Portal for GOSH DRIVE

Our project is to develop an augmented reality (AR) smartphone application aimed at young patients to help ease any worries before they have an operation. The app generates virtual medical equipment in a room which users can interact with and learn about the operation room beforehand. There will also be a web application which doctors can use to build and customise their own rooms to share with patients.

Key Features

- AR room with models that can be interacted with
- Web application that can be used to make customisable rooms

Development Team

UCL 2nd Year Computer Science Students: Yin Ho (Team Leader), Chirag Hegde, Haonan Zhang
