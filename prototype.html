Implementation of Web App Finished Funtionalities

Current functionality of the webapp:
- Create, delete and manage worlds.
- Edit worlds using 2D interface
- Add children and scripts as per functionality of the objects
- Download worlds
- Mobile app integration for sharing worlds

Future web app functionality:
- Search engine for maps
- editing floor, wall and ceiling textures
- (possible) 3D interface for editing maps
- Easy addition of new assets

Implementation of Phone App Finished Funtionalities

Current functionality of the mobile app:
- Place portal on any detected place surface
- User can walk into the portal and explore
- Load room map files and generate the room at runtime
- Play 360 degree video
- Load map files from web server

Future mobile app functionality:
- Local database for storing downloaded worlds
- Track user usage data and send it to analysis
- Voice recognition to interact with objects and UI
- Customizable room layout
